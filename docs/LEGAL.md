## Mobile & stores

- `app.json` → `privacyPolicyUrl`, `termsOfServiceUrl`
- Onboarding and registration link to those URLs
- Play / App Store listing should use the same URLs

## GitHub is public

A public repo **does not** mean open source. The [LICENSE](../LICENSE) states that visibility does not grant use rights. To reduce casual copying you can also:

- Make the repository **private** (strongest for unreleased code)
- Register copyright with the [U.S. Copyright Office](https://www.copyright.gov/) (optional but helps in disputes)
- File a **trademark** for “Closet Org” if you invest in the brand (optional, fee applies)
